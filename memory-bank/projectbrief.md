# Project Brief — Beacon

Beacon is a lightweight project tracker for small product teams. Think "Trello meets a changelog."

- **Who it's for:** 5–50 person product teams.
- **Why it exists:** existing tools are either too heavy (Jira) or too loose (a spreadsheet).
- **Core loop:** create a project → add tasks → ship them → auto-generate a public changelog.
- **Business model:** per-seat SaaS (see [pricing.md](./pricing.md)).
